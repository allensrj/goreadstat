# 安装和使用指南

## 上传到 GitHub

1. 初始化 git 仓库（如果还没有）：
```bash
cd /Users/allensrj/Desktop/project/Achive\ Summary/20260324/goreadstat
git init
git add .
git commit -m "Initial commit: goreadstat v0.1.0"
```

2. 关联远程仓库并推送：
```bash
git remote add origin https://github.com/allensrj/goreadstat.git
git branch -M main
git push -u origin main
```

3. 创建版本标签（推荐）：
```bash
git tag v0.1.0
git push origin v0.1.0
```

## 其他人如何使用

### 方式一：直接安装（推荐）

```bash
go get github.com/allensrj/goreadstat@latest
```

或指定版本：
```bash
go get github.com/allensrj/goreadstat@v0.1.0
```

### 方式二：在项目中使用

在你的 Go 项目中：

```go
// main.go
package main

import (
    "fmt"
    "log"
    "github.com/allensrj/goreadstat"
)

func main() {
    df, meta, err := goreadstat.ReadFile("data.sas7bdat")
    if err != nil {
        log.Fatal(err)
    }
    fmt.Printf("%d rows × %d columns\n", df.RowCount(), df.ColumnCount())
}
```

然后运行：
```bash
go mod init myproject
go mod tidy  # 自动下载依赖
go run main.go
```

## 重要提示

### 用户需要的前置条件：

1. **C 编译器**（gcc 或 clang）
2. **zlib** 库
3. **iconv** 库
4. **make** 工具

### 首次使用步骤：

```bash
# 1. 克隆或下载你的库
go get github.com/allensrj/goreadstat

# 2. 进入库目录构建 ReadStat C 库
cd $GOPATH/pkg/mod/github.com/allensrj/goreadstat@v0.1.0
make

# 3. 在你的项目中使用
```
