# 快速上传到 GitHub

## 1. 初始化并提交

```bash
cd /Users/allensrj/Desktop/project/Achive\ Summary/20260324/goreadstat

git init
git add .
git commit -m "Initial release: goreadstat v0.1.0

- Read/write support for SAS, SPSS, Stata formats
- Column selection feature
- Comprehensive tests and documentation
- CI/CD with GitHub Actions"
```

## 2. 推送到 GitHub

```bash
git remote add origin https://github.com/allensrj/goreadstat.git
git branch -M main
git push -u origin main
```

## 3. 创建版本标签

```bash
git tag -a v0.1.0 -m "Release v0.1.0"
git push origin v0.1.0
```

## 4. 验证

其他人现在可以使用：

```bash
go get github.com/allensrj/goreadstat@v0.1.0
```

完成！
